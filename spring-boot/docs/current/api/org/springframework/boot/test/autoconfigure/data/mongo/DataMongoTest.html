Annotation Type DataMongoTest


  • @Target(value=TYPE)
 @Retention(value=RUNTIME)
 @Documented
 @Inherited

 @ExtendWith(value=org.springframework.test.context.junit.jupiter.SpringExtension.class)
 @OverrideAutoConfiguration(enabled=false)
 @TypeExcludeFilters(value=DataMongoTypeExcludeFilter.class)
 @AutoConfigureCache
 @AutoConfigureDataMongo
 @ImportAutoConfiguration
public @interface DataMongoTest
    Annotation that can be used for a MongoDB test that focuses only on MongoDB components.

    Using this annotation will disable full auto-configuration and instead apply only configuration relevant to MongoDB tests.

    By default, tests annotated with @DataMongoTest will use an embedded in-memory MongoDB process (if available).

    When using JUnit 4, this annotation should be used in combination with @RunWith(SpringRunner.class).
