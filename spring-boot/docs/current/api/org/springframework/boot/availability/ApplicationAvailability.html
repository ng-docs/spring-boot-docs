org.springframework.boot.availability

Interface ApplicationAvailability

  • All Known Implementing Classes:
    ApplicationAvailabilityBean

    public interface ApplicationAvailability
    Provides availability state information for the application.

    Components can inject this class to get the current state information. To update the state of the application an AvailabilityChangeEvent should be published to the application context with directly or via AvailabilityChangeEvent.publish(org.springframework.context.ApplicationContext, S).

    Since:
    2.3.0
    Author:
    Brian Clozel, Phillip Webb
